Enumerator
DEFAULT 

Default state (unused)

CTRL_INIT 

initialization state (when starting the drone)

LANDED

FLYING 

drone flying, doing some move

HOVERING 

drone hovering over a point

TEST 

unknown state

GO_TO_FIX 

drone needing repair (something wrong)

TAKE_OFF 

drone taking off (stays in this state until hovering after take off)

INTERMEDIATE 

state reached between the end of a move and hovering state for instance

LANDING 

drone has begun landing procedure and is not yet landed
